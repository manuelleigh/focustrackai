# Privacidad y Etica

## Principio de uso

FocusTrack AI debe usarse con consentimiento informado. Es un prototipo academico para demostrar tecnologia, no una herramienta lista para sancionar personas ni tomar decisiones laborales sensibles.

## Datos tratados

- Imagen de camara procesada localmente.
- Estado de rostro, ojos, mirada, postura y presencia.
- Aplicacion o ventana activa.
- Historial local de productividad.
- Capturas de pantalla opcionales, desactivadas por defecto.

## Medidas de proteccion

- La aplicacion funciona localmente.
- Los datos se guardan en `data/`.
- Las capturas estan desactivadas por defecto.
- El usuario puede no activar YOLO ni dlib si desea una demo mas liviana.
- El reporte se genera bajo demanda.

## Limitaciones eticas

- No usar el sistema para vigilancia oculta.
- No evaluar desempeno real sin validacion formal.
- No comparar personas sin contexto.
- No conservar capturas innecesarias.
- Informar limitaciones tecnicas: luz baja, lentes, camara mala, rostro cubierto o movimiento rapido.

## Mensaje para sustentacion

El sistema busca apoyar el analisis academico de productividad y distracciones en una demo controlada. Cualquier uso real requeriria politicas internas, consentimiento, revision legal y validacion tecnica independiente.
